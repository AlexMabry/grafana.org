Step 2: Configure

Choose your Configuration Options

The Grafana backend has a number of configuration options defined in it's config file (usually located at /etc/grafana/grafana on linux systems).

In this config file you can change things like the default admin password, http port, grafana database (sqlite3, mysql, postgres), authentication options (google, github, ldap, auth proxy) along with many other options.

Start you grafana server. Login with your admin user (default admin/admin). Open side menu (click the Grafana icon in top menu) head to Data Sources and add your data source.


Step 3: Graph it!

Visualize Your Data

Now that Grafana is installed and configured it's time for the fun stuff: Graphing the data! There a a number of places to get help. You can start by reading the Getting Started guide or checkout the one of the many screencasts.
